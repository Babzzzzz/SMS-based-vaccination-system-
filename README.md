# SMS-based-vaccination-system 
Childhood vaccination remains one of the most effective methods of preventing vaccine-
preventable diseases; however, missed vaccination appointments continue to affect
immunization coverage due to caregiver forgetfulness, inadequate follow-up mechanisms,
and reliance on manual paper-based record-keeping systems. This project focused on the
design and development of an SMS-Based Vaccination Reminder System to improve
vaccination adherence through automated reminders and efficient management of
vaccination records. A review of existing vaccination reminder solutions was conducted to
identify current approaches, strengths, and limitations, which informed the proposed
solution. The project adopted the Object-Oriented Analysis and Design (OOAD) approach
and employed the Waterfall software development methodology to guide the requirements
analysis, system design, development, testing and documentation process. System
requirements were gathered and analysed, after which design artefacts including use case,
class, sequence, entity relationship and database schema diagrams were produced. The
system was developed to enable healthcare providers to register children, create
vaccination schedules, track defaulters, generate reports, and manage vaccination records,
while caregivers receive SMS reminders and respond to appointment notifications. The
system administrator manages users, system settings, and security functions. Functional
and non-functional requirements were incorporated to ensure usability, security, reliability,
and maintainability. System testing was conducted to verify that all major functionalities
operate as intended. The completed system provides an effective and practical solution for
improving vaccination follow-up, reducing missed appointments, and supporting better
immunization management within healthcare facilities.
